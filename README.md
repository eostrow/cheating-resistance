This repository contains all data to create the manuscript figures 
and statistical analyses for the manuscript.

Manuscript title:
Costs of resistance limit the effectiveness of cooperation enforcement in social 
amoebae

Study summary:
This study uses experimental evolution to select for cheating resistance across 
30 replicate populations (=4 cheaters/1 non-cheating control x 6 replicates) of 
Dictyostelium discoideum. Evolved isolates from each population are tested for 
their cross-resistance to other, novel cheaters, as well as trade-offs for fitness
during other stages of the life cycle.

Folders: 
scripts: R Notebook scripts to generate plots and statistics reported in the
manuscript
datafiles: input files for the scripts above
figures: folder for outputing the plots generated by the script (optional)
